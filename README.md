
## Getting Started
1. 生成数据库
   ```shell
    pnpm prisma migrate dev --name init
   ```
   会初始化一条管理员数据，账号密码均为 `admin`

2. Install dependencies:
    ```shell
      pnpm install
    ```
3.  Start the development server:
    ```shell
      pnpm dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
